## 📋 Prasyarat Sistem

Sebelum memulai instalasi, pastikan VPS Anda memenuhi kriteria berikut:

- ✅ **OS**: Ubuntu 20.04 / 22.04 / 24.04
- ✅ **RAM**: Minimal 2GB (direkomendasikan 4GB)
- ✅ **Storage**: Minimal 20GB free space
- ✅ **Domain**: Sudah pointing ke IP VPS (opsional tapi direkomendasikan)
- ✅ **Root Access**: Akses root atau user dengan sudo privileges
